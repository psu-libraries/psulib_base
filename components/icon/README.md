# Icon

Add a PSU Flex Icon.  *Note: only some icons are currently supported.*

## Usage

Default color (black).

```twig
  {% include 'psulib_base:icon' with {
    icon: 'person',
  } %}
```

Specify the color.

```twig
  {% include 'psulib_base:icon' with {
    icon: 'linkOut',
    color: 'link',
  } %}
```
